# 🚀 AI Interview Preparation Assistant

An AI-powered web application that helps job seekers analyze their resumes, improve ATS compatibility, prepare for interviews, and generate professional career documents using Google Gemini AI.

---

## 📌 Features

### 📄 Resume Analysis
- Upload PDF Resume
- Extract Resume Text
- Compare Resume with Job Description
- Resume Match Score
- ATS Score Calculation

### 🧠 Skills Analysis
- Matched Skills
- Missing Skills
- Skill Gap Identification

### 💡 AI Resume Suggestions
- ATS Optimization Tips
- Resume Improvement Suggestions
- Better Keyword Recommendations

### 🎤 Interview Preparation
- AI-generated Interview Questions
- Mock Interview
- AI Feedback on Answers

### 📝 Resume Rewriter
- Rewrite Resume for Better ATS Compatibility
- Improve Professional Language
- Preserve Original Information

### 📄 AI Cover Letter
- Generate Personalized Cover Letter
- Based on Resume & Job Description

### 📊 Professional PDF Report
- Resume Match Summary
- ATS Score
- Skills Analysis
- Resume Suggestions
- Interview Questions
- Mock Interview
- AI Feedback
- Rewritten Resume
- Cover Letter
- Multi-page Professional Report

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/AI-Interview-Preparation-Assistant.git
```

Move into the project

```bash
cd AI-Interview-Preparation-Assistant
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env` file

```env
GOOGLE_API_KEY=YOUR_GEMINI_API_KEY
```

Run the application

```bash
streamlit run app.py
```
